OUR GOALS

  • Keep the great advantages of the NT architecture, while we implement new features and technologies known from other operating systems.
  • Provide flexibility.
  • Compatible with modern Windows drivers and Windows applications.
  • Compatible with modern architecture completely such as x86 and x64 with apps of these architectures to work correctly, and stable.
  • And much more.

    • FEATURES

  • Completely free and open source and community driven modern operating system
  • It is compatible with modern Windows drivers and modern Windows apps!
  • This Project will use code from Wine, and mainly the ReactOS Project and some other projects, the mayority of the code is forked from the ReactOS original repository as the base for ZetroixOS, but we will modify the code by introducing and fixing new features to the Project to make ZetroixOS the stable and modern open source and free operating system compatible with Windows software and drivers in the market (including the open source market)
  • Technical support available directly from our Reddit (/r/zetroixos) and on GitHub Issues
  • And much more coming soon!
